We have a requirement to search business transactions by text in CRM. Specifically, to search service tickets from the agent inbox in IC. Search by Text is not available in standard SAP. We want to enhance the standard SAP search to search by text and use TREX or embedded search to enhance the performance. Created the RFC connection to TREX. We created the serach object connector for the business object BUS2000116_SRVT and did the indexing. Since the templates in the template modeler already have text field, we did not modify the Service Ticket related template.
